# Early-Prediction-of-Cardiovascular-Disease-Using-Clinical-Features
Early Prediction of Cardiovascular Disease Using Clinical Features

**Objective**

The objective of this project is to develop and evaluate a supervised machine learning model that predicts the presence of cardiovascular disease using structured clinical examination data. The aim is to support early detection and risk stratification in clinical settings.

**Dataset Description**

•	Source: Cardiovascular Disease Dataset (https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset)

•	Size: 70,000 patients, 13 variables

•	Features include: demographic variables (age, gender), anthropometric measures (height, weight), clinical measurements (systolic and diastolic blood pressure), biochemical indicators (cholesterol, glucose), and behavioral factors (smoking, alcohol intake, physical activity).

•	Target variable: cardio (0 = no disease, 1 = disease)
